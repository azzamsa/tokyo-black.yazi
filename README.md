## 🎨 Installation

### Manual install

```bash
# Linux/macOS
git clone https://github.com/azzamsa/tokyo-black.yazi.git ~/.config/yazi/flavors/tokyo-black.yazi

# Windows
git clone https://github.com/azzamsa/tokyo-black.yazi.git %AppData%\yazi\config\flavors\tokyo-black.yazi
```

## ⚙️ Usage

Add the these lines to your `theme.toml` configuration file to use it:


```toml
[flavor]
dark = "tokyo-black"
```
